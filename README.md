# Bucket Buddy

Bucket Buddy is a simple Android prototype application for creating and managing a personal bucket list. The app focuses on a clean, straightforward user experience and core functionality without additional configuration or external dependencies.

## Platform

* **Android only**
* **Target SDK:** 26

## Technology

* **Language:** Kotlin

## Features

* Create bucket list items
* Edit existing items
* Delete items from the list
* Organize items using categories
* Simple and intuitive user interface

## Getting Started

### Prerequisites

* Android Studio (any recent version that supports Kotlin and SDK 26)
* Android device or emulator

### Running the App

1. Clone the repository
2. Open the project in **Android Studio**
3. Sync Gradle files
4. Run the app on a device or emulator

No additional configuration is required.

## Project Status

* **Prototype**

The project is intended as a functional prototype and may lack advanced validation, confirmations, or optimizations.
